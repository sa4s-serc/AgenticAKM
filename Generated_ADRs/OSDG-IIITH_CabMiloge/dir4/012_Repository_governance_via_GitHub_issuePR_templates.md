# ADR-012: Repository governance via GitHub issue/PR templates

**Date**: 2025-10-14
**Status**: Proposed

## Context
The repo defines .github templates for issues and pull requests.

## Decision
Standardize contribution and review practices with predefined templates.

## Consequences
- More consistent requirements and change descriptions
- Faster, clearer code reviews and better traceability
- Slight overhead for contributors to fill templates
- Encourages maintainable, auditable change management
