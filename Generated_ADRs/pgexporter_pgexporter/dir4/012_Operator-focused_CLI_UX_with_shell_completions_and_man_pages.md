# ADR-012: Operator-focused CLI UX with shell completions and man pages

**Date**: 2025-10-16
**Status**: Proposed

## Context
Day-2 operations benefit from discoverable commands and consistent tooling.

## Decision
Provide a documented CLI with Bash and Zsh completions and man pages.

## Consequences
Improves operational ergonomics and reduces errors; requires keeping CLI docs and completions in sync with functionality.
