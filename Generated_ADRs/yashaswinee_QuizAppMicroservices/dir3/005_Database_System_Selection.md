# ADR-005: Database System Selection

**Date**: 2025-10-16
**Status**: Proposed

## Context
A stable, feature-rich, and open-source relational database was needed to serve as the persistent data store for the application.

## Decision
PostgreSQL was selected as the relational database management system.

## Consequences
PostgreSQL provides a robust, production-ready database with strong support for data integrity, ACID compliance, and advanced SQL features. Its choice ensures reliability and scalability for the application's data layer and aligns well with the enterprise Java ecosystem.
